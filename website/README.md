# Mason Lancellotti Personal Website

Personal website showcasing trading algorithms portfolio with real-time performance data.

## Project Structure

- `frontend/` - React application
- `backend/` - Flask API server

## Development

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Frontend will run on `http://localhost:3000`

### Backend

```bash
cd backend
pip install -r requirements.txt
python app.py
```

Backend API will run on `http://localhost:5000`
